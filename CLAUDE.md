# Yasno Copy System

Use this repository as the single source of truth for copywriting tasks for the Yasno brand.

For every request involving copy, messaging, naming, headlines, CTAs, support replies, landing pages, emails, bot scripts, or social posts:

1. Read `skills/yasno-copywriter/references/editorial-policy.md`
2. Read `skills/yasno-copywriter/references/tov.md`
3. Read `skills/yasno-copywriter/references/editing-checklist.md`
4. Read `skills/yasno-copywriter/references/ethics-and-restrictions.md`
5. Read `skills/yasno-copywriter/references/terminology.md`
6. Determine the target channel and read the matching file in `skills/yasno-copywriter/references/channels/`
7. Determine the task type and read the matching file in `skills/yasno-copywriter/references/tasks/`
8. If relevant examples exist, read them from `skills/yasno-copywriter/references/examples/`

Then write the draft according to this workflow:

- first identify audience, intent, awareness level, desired action
- propose 2-3 possible angles when useful
- write the text
- run a self-check against the editorial checklist
- remove manipulative, shaming, overpromising, or overly generic phrasing

Always prefer:

- clarity over cleverness
- depth over empty inspiration
- empathy over pressure
- precision over general words
- a real therapeutic tone over pop-psychology cliches

Never write as if therapy guarantees a result, fixes a person, or must be sold through fear, guilt, or shame.
