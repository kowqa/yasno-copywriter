# Editing Checklist

Use this checklist after drafting. It is not optional.

## Language

Ideally, the answer to all of these is `no`:

1. Are there cliches or banalities?
2. Is there bureaucratic language?
3. Is there moralizing or direct instruction?
4. Are there harsh labels or judgmental formulations?
5. Does the text sound pompous?
6. Are there places where rhythm drags?
7. Are there difficult concepts without explanation?
8. Does the text sound all-knowing?
9. Are there unsupported promises?
10. Is there passive voice, too many pronouns, or verbal nouns that make the text heavy?
11. Are there broad claims without facts or grounding?

## Form

Ideally, the answer to all of these is `yes`:

1. Is the text easy to read?
2. Does it start with a hook, without a long preamble?
3. Does the title reflect the real meaning and attract attention honestly?
4. If useful, is there a cultural reference, a wink, or an unexpected entry point?

## Meaning

Ideally, the answer to all of these is `yes`:

1. Can the text be summarized in one thought?
2. Does it work like a small therapeutic session?
3. Does it contain value that is not just generic internet knowledge?
4. Are the main ideas grounded in facts, examples, or clear observations?

## Pleasure

This is the final taste check:

1. Is the text actually interesting?
2. Does it say something fresh?
3. Does it create a thought or feeling worth having?

## Editing rules from the copy desk

- One paragraph, one thought.
- Avoid vague words and duplicated meaning.
- If using bullets, keep grammar parallel.
- Bold only complete thoughts or short concepts that can stand alone.
- Formulate delicately, without loud promises.
- Assume the reader has never seen a Yasno post before.
- If a psychoanalytic term can be avoided, avoid it.
